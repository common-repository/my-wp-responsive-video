== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add a "Video" button on your editor. Use shortcode in page, post or in widgets. 
4. If you want use the WP Responsive Video plugin in your theme php, Place `<?php echo do_shortcode('[youtube]'); ?>` in your templates.
5. In the [youtube] shortcode . 

Shortcodes

<strong>Use WP Video plugin Support Video shortcode</strong>
